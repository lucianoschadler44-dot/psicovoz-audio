# Avatares do PsicoVoz

## Arquivos necessarios

| Arquivo | Descricao |
|---------|-----------|
| shadow_avatar.png | Silhueta misteriosa (avatar inicial) |
| freud_style.png | Ilustracao estilo Freud (psicanalise) |
| skinner_style.png | Ilustracao estilo Skinner (behaviorismo) |
| perls_style.png | Ilustracao estilo Perls (gestalt) |

## Especificacoes

- Formato: PNG com fundo transparente
- Tamanho: 512x512 pixels
- Estilo: Ilustracoes artisticas (nao fotos reais)

## Sugestao de geracao

Use ferramentas como:
- DALL-E / Midjourney / Stable Diffusion
- Prompt exemplo: "stylized illustration portrait of Sigmund Freud, 
  artistic, warm colors, therapeutic, friendly, digital art style"

## Importante

Nao use fotos reais para evitar problemas de direitos de imagem.
Crie ilustracoes estilizadas inspiradas nos psicologos.
